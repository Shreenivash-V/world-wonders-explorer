# 🌍 World Wonders Explorer

A responsive web application that explores famous wonders around the world using the World Wonders API.

## Features

- Fetches live data from the World Wonders API
- Displays wonders in interactive cards
- Search functionality
- Responsive design
- Dynamic content rendering using JavaScript
- GitHub Pages deployment

## Technologies Used

- HTML5
- CSS3
- JavaScript
- REST API
- Git
- GitHub
- GitHub Pages

## API Used

https://www.world-wonders-api.org/v0/wonders/
